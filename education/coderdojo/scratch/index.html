CoderDojo Scratch

    Video Games Built from Scratch

    In the video game module, students learn a visual programming language called Scratch (developed by MIT). Kids learn programming fundamentals and video game basics, building their very own video game in the first few nights of class.

    Website Basics: HTML and CSS

    In the website module, students learn the internet fundamentals, how the browser works, and the basics of how to build something for the web. While gaining a working knowledge of both markup and CSS, kids will craft their very own website. (You’ll have to print it out to put it on the fridge, mom and dad.)
